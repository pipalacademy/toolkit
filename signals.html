Signals¶

toolkit.signals¶

The signal system allows parts of an application to get notified when events occur elsewhere in the application, without a tight coupling.

class toolkit.signals.Signal(name=None)¶

Utility to create a signal and Connect listeners to the signal.

connect(f)¶

Connects a function to this signal.

Can also be used like a decorator

>>> @signal_login.connect
>>> def on_login(username):
>>>     ...
send(*args, **kwargs)¶

Notifies all listeners of this signal.
